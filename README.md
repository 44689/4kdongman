# 4kdongman
手机批量自动化爬取动漫壁纸

# WIKI
第一步:打开Termux 输入  
`apt install python vim git -y`  
第二步:输入  
`git clone https://github.com/44689/4kdongman.git`  
第三步:输入  
`vim main.py`  
修改 第12行代码里的range(1,146)  
`range(开始爬取的页数,结束爬取的页数)`  
然后保存后运行这个程序就自动开始爬取了，保存的目录是当前目录下的4kdongman文件夹里
